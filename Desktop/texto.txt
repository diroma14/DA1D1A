**Información del sistema:**
- `ver` → Versión de Windows
- `date /t` → Ver fecha
- `time` → Ver hora
- `cls` → Limpiar pantalla

**Gestión de directorios y archivos:**
- `mkdir [nombre]` → Crear carpeta
- `rd [nombre] /s /q` → Borrar carpeta
- `cd [ruta]` → Ir a carpeta
- `copy [origen] [destino]` → Copiar archivo
- `move [origen] [destino]` → Mover archivo
- `del [archivo]` → Borrar archivo
- `ren [actual] [nuevo]` → Renombrar archivo

**Listar archivos y carpetas:**
- `dir` → Mostrar contenido
- `dir /p` → Página a página
- `dir /w` → Vista en columnas
- `dir /o:n` → Ordenar por nombre
- `dir /o:-e` → Ordenar por extensión (Z-A)
- `dir /a:d` → Solo carpetas
- `dir /a:-d` → Solo archivos
- `dir /s > [archivo]` → Guardar listado en archivo

**Modificar atributos:**
- `attrib +h [archivo]` → Ocultar archivo
- `attrib -h [archivo]` → Mostrar archivo
- `attrib +r +s [archivo]` → Solo lectura y sistema

**Búsqueda y comparación:**
- `find "texto" [archivo]` → Buscar palabra
- `find "texto" /n /v [archivo]` → Mostrar líneas sin la palabra
- `find "texto" /n /v /i [archivo]` → Ignorar mayúsculas
- `fc [archivo1] [archivo2]` → Comparar archivos

**Copias y organización:**
- `xcopy [origen] [destino] /e /i` → Copiar carpeta completa
- `copy *.txt *.mio` → Copiar todos los `.txt` a `.mio`
- `copy *.mio Todo.mio` → Unir en un solo archivo

**Ordenar archivos:**
- `sort [archivo] /+5` → Ordenar por carácter 5
- `sort [archivo] /+5 /R` → Orden inverso

**Personalizar símbolo del sistema:**
- `prompt $T Nombre:` → Mostrar hora y nombre
- `prompt $P$G` → Restaurar a predeterminado
